# Prompt Pack — Periódico Digital Premium

Este paquete contiene prompts y contexto base para construir un sistema generador y gestor de artículos web para un periódico digital con:

- CMS editorial.
- Artículos públicos y premium.
- Suscripciones.
- Comentarios y reacciones solo con login.
- Publicidad discreta no invasiva.
- Moderación.
- Seguridad, auditoría, SEO y rendimiento.

## Orden recomendado de lectura

1. `SYSTEM_INFO_PERIODICO.md`
2. `PROJECT_BRIEF_PERIODICO.md`
3. `PROMPT_MASTER_PERIODICO.md`
4. `BACKEND_DEVELOPMENT_CONTEXT.md`
5. `FRONTEND_WEB_DEVELOPMENT_CONTEXT.md`
6. `INFRASTRUCTURE_DEVELOPMENT_CONTEXT.md`
7. `PENDIENTES_PERIODICO.md`
8. `CHECKLIST_FINAL.md`

## Regla principal

El sistema debe proteger la reputación del periódico: lectura limpia, publicidad discreta, contenido premium protegido y participación de lectores solo con autenticación.
