# programacionGeneral — Periódico Digital Premium

## Objetivo

Genera código profesional para un periódico digital premium. Prioriza seguridad, claridad, mantenibilidad, experiencia de lectura, protección de contenido premium y control editorial.

## Reglas

- No inventes reglas críticas.
- Documenta pendientes.
- Valida entradas.
- Protege contenido premium.
- Usa permisos granulares.
- Mantén la UI sobria y profesional.
- No uses publicidad invasiva.
- No permitas comentarios/reacciones sin login.
